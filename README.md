# AI SRE Copilot

AI-powered incident management and production debugging platform for DevOps/SRE teams.

## Goal

Build a production-style DevOps project that collects logs, metrics, deployment data, and incidents, then uses AI to:

- Summarize incidents
- Suggest possible root cause
- Recommend debugging steps
- Generate rollback recommendations
- Generate postmortem reports

## DevOps Learning Focus

This project is designed to practice end-to-end DevOps hands-on:

- Linux administration
- Networking fundamentals
- Git and GitHub workflow
- Docker and Docker Compose
- Jenkins CI/CD
- Ansible configuration management
- Terraform infrastructure as code
- AWS cloud deployment
- Kubernetes orchestration
- Helm packaging
- Monitoring with Prometheus and Grafana
- Logging with Loki
- Alerting with Alertmanager
- DevSecOps scanning
- Secrets management
- Rollback strategies
- Production troubleshooting
- AI-powered incident analysis
